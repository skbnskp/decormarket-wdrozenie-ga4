# decormarket-wdrozenie-ga4
Kompletna dokumentacja techniczna wdrożenia Google Analytics 4 (GA4) oraz specyfikacja warstwy danych (DataLayer) dla sklepu decormarket.pl. Repozytorium zawiera wytyczne dla zespołu deweloperskiego (Selly), kody zdarzeń e-commerce oraz audyt stron wymagających poprawy w szablonach.
